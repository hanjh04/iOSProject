#PitchPerfect - Project_1

### 음성을 녹음하고, 다양한 효과를 주어변조한 뒤 재생하는 앱

## 기본 기능

-Rate 조절

-Pitch 조절

-Echo

-Reverb

## 추가 기능

###녹음 화면

-정지 버튼 감추기

-녹음 시간 표시

###재생 화면

-slider 추가. rate, pitch 조절 가능

-한 번에 여러 효과 적용 가능

-시작, 일시정지, 다시 재생, 정지 기능

-오디오 재생 시간과 진행사항 확인
